Normativas para hacer cables de red (Clase A y B)

Los estándares T568A y T568B especifican la disposición de los cables en los conectores RJ45 para cables de red. Aquí tienes una breve descripción de ambos:

1. T568A: En este estándar, los pares de cables naranja y verde se cruzan. Es ideal para redes domésticas y pequeñas empresas.

2. T568B: Aquí, los pares de cables naranja y verde están en posiciones diferentes. Se prefiere en entornos empresariales y de alta demanda.

Para un cable directo, utiliza el mismo estándar en ambos extremos (T568A a T568A o T568B a T568B). Si necesitas un cable cruzado, combina ambos estándares (T568A a T568B).

¿Para que se utiliza un cable directo en la red?

1. Conectar una computadora a un switch o router: Si deseas conectar una computadora o dispositivo final (como una impresora) a un switch o router, debes usar un cable directo. En este caso, ambos extremos del cable deben seguir el mismo estándar (T568A o T568B).

2. Conectar un switch a un router o módem: Cuando conectas un switch a un router o módem, también necesitas un cable directo. Esto permite que los dispositivos se comuniquen correctamente.

3. Conectar dos computadoras directamente: Si deseas transferir archivos entre dos computadoras sin usar un switch o router, un cable directo es necesario. Esto crea una conexión punto a punto entre las computadoras.

¿Para que se utiliza un cable cruzado en la red?

1. Conectar dos computadoras directamente: Si deseas transferir archivos entre dos computadoras o establecer una conexión punto a punto, un cable cruzado es necesario. Ambas computadoras pueden comunicarse sin problemas.

2. Conectar dos switches o hubs: Cuando necesitas expandir tu red local y conectar dos switches o hubs, un cable cruzado permite que los dispositivos se comuniquen entre sí.

3. Conectar dos routers: En algunos casos, como configuraciones específicas de laboratorio o pruebas, puedes usar un cable cruzado para conectar dos routers directamente.
