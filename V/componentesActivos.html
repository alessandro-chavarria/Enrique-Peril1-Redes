SWITCH
¿QUE ES?
Un switch es un dispositivo de red que conecta múltiples dispositivos dentro de una red local (LAN). Su función principal es reenviar tráfico de datos entre los dispositivos conectados. Los switches operan en la capa 2 (capa de enlace de datos) del modelo OSI. Pueden ser gestionables o no gestionables, según si permiten configuración y monitoreo.
ROUTER
¿QUE ES?
Un router es un dispositivo que conecta redes diferentes (como una red local a Internet) y dirige el tráfico entre ellas. Opera en la capa 3 (capa de red) del modelo OSI. Realiza funciones como enrutamiento, NAT (traducción de direcciones de red) y filtrado de paquetes.
TARJETA DE RED
¿QUE ES?
Una tarjeta de red (NIC, por sus siglas en inglés) es un componente que permite que una computadora se conecte a una red. Puede ser una tarjeta física (como una tarjeta Ethernet) o una interfaz inalámbrica (como Wi-Fi).
INTERRUPTOR INTELIGENTE WI-FI
¿QUE ES?
Un interruptor inteligente Wi-Fi es un dispositivo que se conecta a la red inalámbrica y permite controlar dispositivos eléctricos (como luces o electrodomésticos) a través de una aplicación móvil o comandos de voz.
ACCESS POINT
¿QUE ES?
Un access point es un dispositivo que proporciona conectividad inalámbrica a una red cableada. Permite que dispositivos como teléfonos, tabletas y computadoras se conecten a la red Wi-Fi.
